# AGENTS.md

本文件用于约束 AI 在本仓库中的协作方式。

## 目标
- 以最小改动完成需求，优先保证脚本稳定性与可维护性。
- 修改后给出清晰结论：改了什么、为什么改、影响范围。

## 命令与检索偏好
- 优先使用 `rg`，不要默认使用 `Get-Content`。
- 常用方式：
  - 列出文件：`rg --files`
  - 搜索文本：`rg -n "pattern" <path>`
  - 带上下文搜索：`rg -n -C 3 "pattern" <path>`
- 仅在必须完整查看文件原文时使用 `Get-Content`。

## 代码修改约定
- 优先做小步修改，不做无关重构。
- 不随意改动已有注释、命名和代码风格。
- 若发现潜在回归风险，先指出风险点再实施修改。

## 输出要求
- 默认使用中文，简洁直接。
- 先给结果，再给关键细节。
- 引用文件时给出明确路径与必要行号。
